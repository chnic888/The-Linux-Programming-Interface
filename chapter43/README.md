# INTERPROCESS COMMUNICATION OVERVIEW

## A Taxonomy of IPC Facilities
- `communication` 关注process之间数据的交换
- `synchronization` 关注process或thread操作之间的同步
- `signals` 在某种情况下可作为一种同步技术，也可以通过在realtime signal上绑定数据来作为通信技术
